
  # High-Fidelity Budgeting App Design

  This is a code bundle for High-Fidelity Budgeting App Design. The original project is available at https://www.figma.com/design/zsLWdVePPgVtHcBzQnsQ25/High-Fidelity-Budgeting-App-Design.

  ## Running the code

  Run `npm i` to install the dependencies.

  For Supabase integration, copy `.env.example` to `.env` and fill:
  - `VITE_SUPABASE_URL`
  - `VITE_SUPABASE_ANON_KEY`

  If these variables are missing, the app runs in local-only mode.

  Run `npm run dev` to start the development server.
